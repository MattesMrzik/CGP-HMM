
## Install

contact mattes@mrzik.de

* git clone --recursive https://github.com/MattesMrzik/CGP-HMM
* cd viterbi_cc
* make
* chmod u+x Viterbi

## Run

* navigate to cgp_hmm
* cd src
* python3 cgphmm.py
